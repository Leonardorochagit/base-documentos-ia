# Sobre

O **Base Documentos IA** foi criado para reunir artigos técnicos sobre organização documental, consulta com inteligência artificial, ingestão de arquivos e desenvolvimento de soluções em Python e VS Code.

A proposta do portal é simples: transformar conhecimento solto em uma base mais útil, navegável e conectada a fluxos reais de trabalho.
